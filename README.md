# Bloonsplosion
Un Asteroids mais 
avec des cercles à la place.

Contrôle : 
Z : avancer
Q : tourner à gauche
D : tourner à droite
Espace : tirer

V0 : fonctionnel et 
utile en cas de problème

V0.1 : fonctionnel,
tentative de rajouter
des rebondissement 
entre les cercles.
Elle contient 
seulement le 
calcul des 
nouvelles
vélocités
après une
collision.
